Detailed Description

Handler for a list of ExecuteGroupSecondary that are part of the same graph group. All the execute groups share the same renderpass and each of the groups correspond to a subpass of the renderpass. Also each execute group uses one or more sub encoders to record their work. One parallel encoder will be used for creating the sub encoders of each execute group.

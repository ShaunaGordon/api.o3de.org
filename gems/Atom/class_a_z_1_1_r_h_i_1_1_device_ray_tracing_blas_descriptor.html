Detailed Description

DeviceRayTracingBlasDescriptor

The Build() operation in the descriptor allows the BLAS to be initialized using the following pattern:

RHI::DeviceRayTracingBlasDescriptor descriptor; descriptor.Build() ->Geometry() ->VertexFormat(RHI::Format::R32G32B32_FLOAT) ->VertexBuffer(vertexBufferView) ->IndexBuffer(indexBufferView) ;

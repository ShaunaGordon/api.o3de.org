Detailed Description

template<typename GeometryViewType, typename StreamBufferViewType>
class AZ::RHI::StreamIterator< GeometryViewType, StreamBufferViewType >

Helper class to conveniently iterate through stream buffer views using the StreamBufferIndices class above Takes as input GeometryView* and StreamBufferIndices, then can be used to iterate with ++ operators It also supports direct indexing via myStreamIter[idx] syntax. This class is templated to support both GeometryView and DeviceGeometryView

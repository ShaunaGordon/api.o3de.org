Detailed Description

DeviceRayTracingTlasInstance

Each TLAS instance entry refers to a DeviceRayTracingBlas, and can contain a transform which will be applied to all of the geometry entries in the Blas. It also contains a hitGroupIndex which is used to index into the DeviceRayTracingShaderTable to determine the hit shader when a ray hits any geometry in the instance.

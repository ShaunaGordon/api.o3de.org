Detailed Description

DeviceRayTracingShaderTableDescriptor

The Build() operation in the descriptor allows the shader table to be initialized using the following pattern:

RHI::DeviceRayTracingShaderTableDescriptor descriptor; descriptor.Build(AZ::Name("RayTracingExampleShaderTable"), m_rayTracingPipelineState) ->RayGenerationRecord(AZ::Name("RayGenerationShader")) ->MissRecord(AZ::Name("MissShader")) ->ShaderResourceGroup(missSrg) ->HitGroupRecord(AZ::Name("HitGroup1")) ->ShaderResourceGroup(hitGroupSrg1) ->HitGroupRecord(AZ::Name("HitGroup2")) ->ShaderResourceGroup(hitGroupSrg2) ;
